# Prime Autohaus - Car Dealership Website

A clean, responsive car dealership website built with HTML, CSS, and JavaScript.

## Features

- Professional homepage with full-width automotive hero image
- Featured vehicle inventory
- Search, type filter, and max-price filter
- Finance payment estimator
- Services section
- Customer note section
- Test drive contact form
- Responsive navigation for mobile screens

## How to Run

Open `index.html` in a browser.

No installation or build step is required.
